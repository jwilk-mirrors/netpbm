NAME

xbmtopbm - convert an X11 or X10 bitmap to a PBM image

SYNOPSIS

xbmtopbm [bitmapfile]

DESCRIPTION

This program is part of Netpbm.

xbmtopbm reads an X11 or X10 bitmap as input and produces a PBM image as output.
