NAME

atktopbm - convert Andrew Toolkit raster object to PBM

SYNOPSIS

atktopbm [atkfile]

DESCRIPTION

This program is part of Netpbm.

atktopbm reads an Andrew Toolkit raster object as input. and produces a PBM image as output.

OPTIONS

There are no command line options defined specifically for atktopbm, but it recognizes the options common to all programs based on libnetpbm (See Common Options.)
