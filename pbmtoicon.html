NAME

pbmtoicon - convert a PBM image into a Sun icon

SYNOPSIS

pbmtoicon [pbmfile]

DESCRIPTION

This program is part of Netpbm.

pbmtoicon reads a PBM image as input and produces a Sun icon as output.
