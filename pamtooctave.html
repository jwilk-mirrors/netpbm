NAME

pamtooctave - convert a Netpbm image to a GNU Octave image

SYNOPSIS

pamtooctave [netpbmfile]

DESCRIPTION

This program is part of Netpbm.

pamtooctave reads a Netpbm image as input and produces a GNU Octave image file as output. An Octave image file is an ASCII representation of two matrices:

  1. the image itself as a list of indexes into a colormap, and
  2. the corresponding colormap as a list of {red, green, blue} triplets.

netpbmfile is the name of the file containing the input PNM or PAM image, or - to indicate Standard Input. If you don't specify netpbmfile, the input is from Standard Input. pamtooctave converts only the first image in the input stream. pamtooctave writes the output Octave image to Standard Output.

NOTES

There is no octavetopam program. However, GNU Octave's saveimage command can save images in PPM format.

HISTORY

pamtooctave was new in Netpbm 10.39 (June 2007).
